# Skills Sphere

An interactive online learning platform designed to help users browse, discover, and enroll in various development and tech courses.

## Live URL
[Skills Sphere Live URL](https://skills-sphere-seven.vercel.app/)

## Key Features
* **Course Catalog:** Browse a wide range of courses categorized by level and topic.
* **Dynamic Course Details:** View in-depth details about each course including instructor, duration, and rating.
* **Course Curriculum:** Interactive, responsive modules for each course to track learning progress.
* **Responsive Design:** Optimized for mobile and desktop screens using Tailwind CSS.
* **Server-side Rendering & Routing:** Built with Next.js 15+ for fast performance.

## Tech Stack & npm Packages Used
* `next JS`: React framework for production
* `HeroUI`: A component library/plugin for Tailwind CSS that provides clean utility classes for components.
* `BetterAuth`: A comprehensive and secure authentication library designed for Next.js
* `tailwindcss`: Utility-first CSS framework
